# Healthcare Data Cleaning Project

## Overview
This project focuses on cleaning and preprocessing a healthcare dataset using Python in Google Colab.

## Tasks Performed
- Handled missing values
- Fixed inconsistent values
- Detected and removed outliers
- Performed data quality checks
- Prepared clean dataset for analysis

## Tools Used
- Python
- Pandas
- NumPy
- Google Colab

## Project Objective
To improve dataset quality by applying data cleaning techniques for accurate analysis.
